我的LinuxSecurity书单

2018-09-17 10:31
By Panda Guo

从事开发工作20年了， 在过往软件开发过程中， 陪伴最多的是书，有些书值得好好读读。 最近经常被兄弟们问道有些哪些书值得推荐。因此整理了一个读过的书单，且感觉读完还有收获的推荐给大家。

另， 在整理书单过程中， 有些书有新的版本发布，我就列出当前的最新版本。

好东西当然要拿出来分享^_^!

计算机及系统原理

  • 《深入理解计算机系统（原书第3版） 》

    • [美] 兰德尔 E.布莱恩特（Randal E.·Bryant） 著，龚奕利，贺莲 译
    • 出版社： 机械工业出版社

  • 《Linux内核设计与实现（原书第3版） 》

    • [美] Robert Love 著，陈莉君，康华 译
    • 出版社： 机械工业出版社

这本书，我读了两版， 第二版，第三版我都买了。

  • 《O'Reilly：深入理解LINUX内核（第3版）》
    • [美] 博韦，西斯特 著，陈莉君，张琼声，张宏伟 译
    • 出版社： 中国电力出版社

兄弟我2007年由津入京工作， 在回天津图书馆退卡，由押金换购的书，就是这本。。

  • 《O'Reilly：LINUX设备驱动程序（第3版） 》
    • [美] 科波特（Corbet J.） 著，魏永明，耿岳，钟书毅 译
    • 出版社： 中国电力出版社

搞linux下驱动，人手必备。

  • 《鸟哥的Linux私房菜 （基础学习篇 第三版） 》
    • 鸟哥 著，王世江 编
    • 出版社： 人民邮电出版社

编程开发

系统平台

  • 《UNIX环境高级编程（第3版） [Advanced Programming in the UNIX Environment, Third Edition]》
    • [美] W. Richard Stevens，[美] Stephen A. Rago 著，戚正伟，张亚英，尤晋元 译
    • 出版社： 人民邮电出版社

Unix下的编程圣经， 其实关于“W.Richard Stevens”的书，不读过，怎么敢说是混unix（linux）圈的。

  • 《Linux程序设计（第4版） [Beginning Linux Programming 4th Edition]》

    • [英] 马修（Matthew N.） 著，陈健，宋健建 译
    • 出版社： 人民邮电出版社

  • 《UNIX网络编程 卷1 套接字联网API（第3版）》

    • [美] W.Richard Stevens，[美] Bill Fenner，[美] Andrew M. Rudoff 著
    • 出版社： 人民邮电出版社

UNIX和网络专家W. Richard Stevens的传世之作

  • 《UNIX网络编程 卷2 进程间通信（第2版） 》
    • [美] W.理查德·史蒂文斯（W.Richard Stevens） 著
    • 出版社： 人民邮电出版社

UNIX和网络专家W. Richard Stevens的传世之作

编程语言

C

  • 《C程序设计语言》
    • 美国）Brian W. Kernighan，Dennis M. Ritchie
    • 出版社： 机械工业出版社

由C语言的设计者Brian W.Kernighan和Dennis M.Ritchie编写的一部介绍标准C语言及其程序设计方法的权威性经典著作， 有这句话就够了。

  • 《C语言程序设计：现代方法（第2版）》
    • [美] 金 著，吕秀锋，黄倩 译
    • 出版社： 人民邮电出版社

这本书作为《C程序设计语言》的补充， 介绍内容更为详细。

一下几本， 可作为上述两本书的补充。。。。。

  • 《C和指针 POINTERS ON C 》
    • [美] 里科（Kenneth·Reek） 著，徐波 译
    • 出版社： 人民邮电出版社

C 之所以难， 在于指针； C之所以值得推荐， 在于C指针。 看完这本，对于指针的问题，一切迎刃而解。

  • 《C陷阱与缺陷 [C Traps and Pitfalls]》

    • [美] Andrew Koenig 著，高巍 译
    • 出版社： 人民邮电出版社

  • 《C专家编程Expert C Programming Deep C Secrets 》

    • [美] 林登（LinDen P.V.D） 著，徐波 译
    • 出版社： 人民邮电出版社

  • 《Linux C编程一站式学习》

    • 北京亚嵌教育研究中心组,宋劲彬著
    • 出版社：电子工业出版社

这本书，好像买不到了。 我当年在亚嵌时，使用的课程讲义。也是我在研究C上，花时间最多的一本书。

C++

  • 《C++编程思想》 原书第二版
    • [美] Bruce Eckel，Chuck Allison 著，刘宗田 等 译
    • 出版社： 机械工业出版社

我2002年春节期间， 在南开大学读过的，第一本C++的书，引为经典。。

  • 《C++ Primer中文版》 第5版
    • Stanley B. Lippman，Josée Lajoie，Barbara E. Moo 著，王刚，杨巨峰 译，叶劲峰，李云，刘未鹏 等 校
    • 出版社： 电子工业出版社

我读的是侯捷老师译的C++ Primer版本。

ASM

  • 《汇编语言程序设计|》
    • 美 Richard Blum 著，马朝晖 等 译
    • 出版社: 机械工业出版社

这是一本讲解 at&t格式的汇编语言，基于X86体系，适用于Unix & 类Unix 平台

Shell

  • 《O'Reilly：Shell脚本学习指南》 [美] 罗宾，[美] 比博 著，O'Reilly Taiwan公司 译
    • O‘Reilly的书，只要和自己对口，都值得收藏。
    • 出版社： 机械工业出版社

Python

  • 《Python编程 从入门到实践 》

    • [美] 埃里克·马瑟斯（Eric Matthes） 著，袁国忠 译
    • 出版社： 人民邮电出版社

  • 《Python绝技：运用Python成为顶级黑客 》

    • [美] TJ O'Connor（TJ 奥科罗） 著，崔孝晨，武晓音 译
    • 出版社： 电子工业出版社

  • 《Python 黑帽子：黑客与渗透测试编程之道 》

    • [美] Justin Seitz（贾斯汀·塞茨） 著，孙松柏，李聪，润秋 译
    • 出版社： 电子工业出版社

算法

  • 《算法导论（原书第3版）》

    • [美] Thomas H.Cormen，[美] Charles E.Leiserson，[美] Ronald L.Rivest，[美] Clifford Stein 著，殷建平，徐云，王刚 等 译
    • 出版社： 机械工业出版社

  • 《算法：C语言实现（第1-4部分）》 基础知识、数据结构、排序及搜索（原书第3版） [Algorithms in C, Parts 1-4: Fundamentals, Data Structures, Sorting, Searching,Third Edition]

    • [美] 塞奇威克 著，霍红卫 译
    • 出版社： 机械工业出版社

网络技术

  • 《TCP/IP详解 卷1：协议（原书第2版）》
    • [美] 凯文R.福尔（Kevin R. Fall，W.Richard Stevens） 著，吴英，张玉，许昱玮 译，吴功宜 校
    • 出版社： 机械工业出版社

Stevens经典网络名著的整体重组和彻底更新， 值得一读

  • 《Wireshark数据包分析实战（第2版） 》
    • [美] Chris Sanders 著，诸葛建伟，陈霖，许伟林 译
    • 出版社： 人民邮电出版社

搞网络的，不懂嗅探怎么行， 做为wireshark的入门书，值得一读。

  • 《深入理解Linux网络技术内幕 》
    • [意] Benvenuti 著，夏安，闫江毓，黄景昌 译
    • 出版社： 中国电力出版社
